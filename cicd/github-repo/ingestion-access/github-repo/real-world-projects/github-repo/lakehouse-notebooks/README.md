# Lakehouses & Notebooks

Working with Fabric Lakehouses, Delta Lake, and PySpark notebooks — covering ingestion patterns, transformations, joins, window functions, and data quality at scale in Spark.

## Topics

- **Lakehouse fundamentals** — architecture, OneLake mechanics, Delta Lake internals, tables and schemas
- **PySpark fluency** — DataFrames, filtering, GroupBy, joins (essentials + edge cases), window functions, complex types, data formatting
- **Data cleaning at scale** — applied PySpark transformation on real Audible audiobook data
- **Incremental loading** — production patterns for table updates inside the Lakehouse
- **Notebook orchestration** — running notebooks within Data Pipelines, with Python package management
- **Data quality in Spark** — DataFrame-level validation patterns
- **Local development** — Fabric Data Engineering VS Code extension, AI-assisted dev workflows
- **Operational details** — SQL Endpoint force refresh for stale-metadata edge cases
