# Platform, Security & Governance

Administering Fabric — capacity management, the full data security stack (RLS / OLS / CLS, Dynamic Data Masking, RBAC), service principals, and capacity automation via API.

## Topics

- **Access & identity** — Trusted Workspace Access patterns, item creation via Service Principals for automation-friendly identity
- **Data security stack** — Row-Level Security (RLS), Object-Level Security (OLS), Column-Level Security (CLS), Dynamic Data Masking in Data Warehouse, and Lakehouse RBAC
- **Capacity model** — F-SKU mechanics, creating capacities, capacity metrics, and the trade-offs that drive sizing decisions
- **Capacity operations** — throttling avoidance, FUAM (Fabric Unified Admin Monitoring) for visibility, capacity automation via REST API
- **Capacity Consumption Challenge** — applied optimisation exercise
