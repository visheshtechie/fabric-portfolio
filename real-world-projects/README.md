# Real-world Projects

The portfolio's headline work — two end-to-end Fabric implementations delivered sprint-by-sprint, each going from architecture and capacity design through ingestion, transformation, orchestration, validation, and go-live readiness.

This is where the foundations from every other module come together into production-shaped builds.

## Project 1 — Standard tier

Sprint-based delivery of a complete Fabric data platform across seven sprints: capacity & workspace design, version control & deployment, ingestion, Lakehouse development, transformation & movement, orchestration & data validation, and go-live preparation.

## Project 2 — Advanced tier

Higher-complexity build with deeper CI/CD automation, REST API ingestion (YouTube), and a metadata-driven processing framework. Eight sprints culminating in a complete solution walkthrough.

Coverage:
- Capacity & workspace topology designed for environment isolation
- Full CI/CD automation across dev → test → prod
- YouTube REST API as the source data
- Lakehouse-based medallion architecture (bronze → silver → gold)
- Metadata-driven processing framework
- Production go-live readiness and final solution walkthrough

This is the centrepiece project for the portfolio — the most complete demonstration of applied Fabric data engineering across the full lifecycle.
