# Ingestion & Access

Getting data into Microsoft Fabric from flat files, REST APIs, and external databases. Covers the full ingestion toolkit — Dataflow Gen2, Copy Jobs, OneLake Shortcuts, and Database Mirroring — with a focus on incremental load patterns and production-grade extraction.

## Topics

- **Dataflow Gen2** — ETL vs ELT design, scheduling, data type handling, cleansing, reshaping, and Power Query M for transformations
- **Incremental loading** — Copy Job Merge, Copy Data Activity Upsert, and Spark Merge approaches, with trade-off analysis
- **REST API ingestion** — generic 8-step extraction framework applied to real APIs (GitHub)
- **Zero-copy access** — OneLake Shortcuts and Database Mirroring strategies for accessing external data without duplication
- **Real-world transformations** — the three-part Restaurant Data Detective project, exploring Fast Copy and Staging effects on Dataflow performance
