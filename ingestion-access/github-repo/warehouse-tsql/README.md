# Data Warehouse & T-SQL

Building dimensional warehouses in Fabric — foundational T-SQL through Kimball modelling, watermark patterns, and a multi-sprint real-world warehouse project (CONFX) implementing both SCD Type 1 and Type 2.

## Topics

- **T-SQL fundamentals** — SELECT, WHERE, GROUP BY, JOINs, subqueries, CTEs, INSERT/UPDATE/DELETE, CTAS, ALTER, TRUNCATE
- **Warehouse objects** — schemas, tables, views, stored procedures, IF/ELSE conditional logic
- **Dimensional modelling theory** — OLTP vs OLAP, normalisation (1NF/2NF/3NF), Kimball star schema, surrogate keys, slowly changing dimensions
- **CONFX project** — end-to-end conference data warehouse with sessions, ticket sales, conformed date dimension, attendance fact table, and SCD Type 1 + Type 2 implementations
- **Watermarking** — control-table-based incremental load tracking with stored procedures, integrated into Data Pipelines
- **Tooling** — connecting to the Fabric warehouse via SSMS for richer development workflows
